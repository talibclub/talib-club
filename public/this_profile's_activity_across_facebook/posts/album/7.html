#QA กฎของการเฉลิมฉลองวันฮิญาญสากล คำถาม : วันที่ 1 กุมภาพันธ์ ในทุกๆปี เป็นวันฮิญาบของโลก อะไรคือความคิดเห็นของบรรดาอุลามาอฺต่อประเด็นนี้? แล้วชัยคฺคิดอย่างไรบ้างต่อประเด็นนี้? สิ่งนี้เป็นบิดอะฮฺด้วยหรือไม่? คำตอบ : อัลฮัมดุลิลละฮฺ วันฮิญาบของโลกถูกคิดโดยผู้หญิงมุสลีมะฮฺท่านหนึ่งที่อยู่ในประเทศอเมริกา มีชื่อว่า นัซมะฮฺ คาน เขาเป็นมุสลิมะฮฺจากเชื้อสายบังคลาเทศ อพยพมาอยู่อเมริกาเมื่ออายุ 11 ปี ที่นั่นเขาได้เจอกับการล่วงละเมิดทางการปฎิบัติเพียงเพราะฮิญาบของเธอ เธอจึงคิดว่าจะจัดการต่อต้านการถูกเหยียดเชื้อสายต่อฮิญาบอย่างนี้อย่างไร เธอจึงเรียกร้องมุสลีมะฮฺทุกประเทศ ทุกศาสนา และทุกกลุ่ม สวมฮิญาบในวันที่ 1 กุมภาพันธ์ ซึ่งต่อมาได้รู้จักกันในวันฮิญาบโลก ฮิญาบเป็นสิ่งวาญิบที่อัลลอฮฺ ( ซุบฮานะฮูวาตะอาลา ) ได้บัญญัติไว้สำหรับมุสลีมะฮฺผู้ที่ศรัทธาต่อพระองค์อัลลอฮฺ ฮิญาบคือสัญลักษณ์ของความเคารพตนเองและความตักวา เป็นข้อบังคับสำหรับชาวมุสลิม อูลามาอฺ นักดาอีย์ และทุกๆคน ที่จะต้องระดมพลังและความสามารถในการเชิญชวนกระตุ้นเพื่อสวมฮิญาบ อย่างไรก็ตาม ถึงกระนั้น วิธีการที่ใช้ควรเป็นไปตามวิธีที่กำหนดไว้ เพราะเป้าหมายที่ดีต้องเป็นไปตามแนวทางที่กำหนด สำหรับการกำหนดวันหนึ่งในหนึ่งปีและมีการรำลึกซ้ำๆทุกปี และเรียกมันว่าวันฮิญาบโลกนั้นเป็นสิ่งที่ไม่อนุญาต ด้วยเหตุผลดังต่อไปนี้ 1. มันคือการเลียนแบบประเพณีของกาเฟรที่เป็นศัตรูต่อพระองค์อัลลอฮฺ และเป็นศัตรูต่อบรรดารอซูลของพระองค์ เป็นผู้เสนอแนวคิดดังกล่าวและจัดสรรสำหรับการส่งเสริมให้เกิดการเฉลิมฉลองซ้ำๆ เช่น วันเด็กสากล วันต่อต้านความรุนแรงต่อสตรีสากล วันมะเร็งสากล เป็นต้น ของบิดอะฮฺและการปฎิบัติที่ชั่วร้ายที่พระองค์อัลลอฮฺไม่ได้สั่งใช้ แต่สิ่งนี้มีแนวโน้มเป็นบิดอะฮฺที่น่าตำหนิมากกว่าเพราะการจัดสรรวันใดวันหนึ่งซ้ำๆกันทุกปี ซึ่งผู้คนมารวมตัวกันทำให้วันนี้เป็นวันอีด( รากศัพท์คือสิ่งที่กลับมาทำซ้ำ ) บรรดานักวิชาการในสถาบัน ได้กล่าวใน Fatwa lajnah Da'imah Vol.1 (33:8) ว่า : อีดเป็นคำที่มีความหมายว่า การกลับมาทำซ้ำของประเพณี ปีละครั้ง เดือนละครั้ง และอื่นๆ เช่น วันอีด วันญุมอะห์( วันศุกร์ ) เป็นต้น ข้อกำหนดของวันอีดต้องมาจากพระองค์อัลลอฮฺเท่านั้น พระองค์อัลลอฮได้ทรงกำหนดวันอีดให้แก่มุสลิมแค่ 2 วัน วันอีดอัฎฮา และวันอีดฟิตรี และจะมีอีดประจำสัปดาห์คือวันญุมอะฮฺ วันอีดอื่นที่ถูกสร้างขึ้นมาซึ่งไม่ได้เป็นที่รู้จักกันในยุคต้นที่ชอบธรรมนั้น มันอาจจะมีต้นกำเนิดจากผู้ที่ไม่ใช่มุสลิม ดังนั้นพวกนี้จึงเข้าค่ายการเลียนแบบศัตรูของอัลลอฮฺ ฉะนั้นมันจึงเป็นบิดอะฮฺ ( จากมัจมุ ฟัตวา อิบนุ ตัยมียะห์ 2/301 ) 2. มันเป็นสิ่งที่ตรงกันข้ามกับหลักชารีอะฮฺโดยสิ้นเชิงที่เกี่ยวกับชารีอะฮฺฮิญาบ ทั้งในทุกๆด้านของชารีอะฮฺ เพราะชะรีอะฮฺของอัลลอฮฺ ( ซุบฮานะฮูวาตะอาลา ) นั้นจะต้องถูกปฏิบัติทันทีและเชื่อฟังด้วยการนอบน้อมอย่างเต็มที่และหวังผลตอบแทนจากอัลลอฮฺ ( ซุบฮานะฮูวาตะอาลา ) และแรงจูงใจในการทำอิบาดะฮฺในขณะที่หวังผลตอบแทนจากอัลลอฮฺ ( ซุบฮานะฮูวาตะอาลา ) และกลัวการลงโทษของพระองค์ สำหรับสตรีจำนวนหนึ่งที่รวมตัวกันในวันฮิญาบสากลในขณะที่พวกเธออยู่ในอาการดีใจและสนุกสนาน พวกเขาเชิญชวนให้สตรีจากศาสนาและกลุ่มชาติพันธุ์ต่างๆ สวมฮิญาบเพียงหนึ่งวัน แล้วถ่ายรูปเผยแพร่ ภาพถ่ายดังกล่าวเป็นรูปแบบหนึ่งของการโฆษณาชวนเชื่อและหลังจากนั้น หากเผยแพร่ออกไปอีก แสดงว่าเป็นการเล่นกับบทบัญญัติของอัลลอฮฺ เพราะการคลุมฮิญาบเป็นการเคารพบูชาที่ต้องตั้งอยู่บนพื้นฐานของความตั้งใจ และการหวังผลตอบแทน และต้องมีความต่อเนื่อง 3. ฮิญาบที่ผู้หญิงสวมใส่ในวันนั้นสร้างภาพลบมากมายต่อบทบัญญัติของอัลลอฮฺ ( ซุบฮานะฮูวาตะอาลา ) ข้อเสียมีมากกว่าประโยชน์ เนื่องจากการคลุมฮิญาบตามชารีอะฮฺนั้นจะต้องมีเงื่อนไขและหลักเกณฑ์ของมัน หากมีทั้งหมดแล้ว มันก็ย่อมเป็นฮิญาบตามแบบชารีอะห์ที่อัลลอฮฺ ( ซุบฮานะฮูวาตะอาลา ) ได้สั่งใช้สำหรับผู้หญิง แต่อย่างไรก็ตาม หากไม่มีเงื่อนไขเหล่านี้หรือบางเงื่อนไขไม่มีอยู่ มันก็ไม่ถูกเรียกว่า ชารีอะฮฺฮิญาบ หลักเกณฑ์เหล่านี้ได้ถูกอธิบายไว้ในฟัตวาเลขที่ 6991 สำหรับฮิญาบที่ผู้หญิงมักจะสวมใส่ในงานเฉลิมฉลองโดยทั่วไปแล้วนั้นมันจะไม่ตรงตามฮิญาบที่ถูกต้อง อันที่จริง ผู้หญิงคนนั้นไม่ได้ทำอะไรเลยนอกจากปกปิดผมและร่างกายของเธอ นอกจากนี้ เธอยังสวมกางเกงขายาว บางครั้งก็สวมเสื้อผ้าที่รัดรูปและโชว์ส่วนเว้าส่วนโค้ง บางครั้งก็แต่งหน้าสวย บางครั้งก็สวมเสื้อผ้าในรูปแบบของเครื่องประดับที่มีสีสันและประกายระยิบระยับดึงดูดสายตาและเคลื่อนไหวให้หัวใจอ่อนแอ ทั้งหมดนี้เป็นการขัดต่อฮิญาบที่อัลลอฮฺ ( ซุบฮานะฮูวาตะอาลา ) ทรงบัญญัติไว้ ด้วยเหตุนี้ การเฉลิมฉลองดังกล่าว ที่เรียกกันว่า "วันฮิญาบโลก" จึงไม่ได้รับอนุญาต แม้ว่าความตั้งใจของผู้อยู่เบื้องหลังจะดีและจริงใจก็ตาม และอย่างไรก็ตาม ความตั้งใจจริงอย่างเดียวไม่เพียงพอ แต่จำเป็นอย่างยิ่งที่วิธีการที่ใช้ควรเป็นไปตามคำสอนของอิสลาม และไม่เกี่ยวข้องกับสิ่งที่ขัดต่อคำสั่งของอัลลอฮฺ ขอพระองค์ทรงได้รับเกียรติ อย่างไรก็ตาม หากกลุ่มมุสลิมชายหรือหญิงรวมตัวกันในสถานที่ใดเวลาหนึ่งเพื่อส่งเสริมภาระหน้าที่ของฮิญาบและแจ้งให้ผู้คนทราบ นี่ก็เป็นสิ่งที่ดีและอยู่ภายใต้การส่งเสริมสิ่งที่ถูกต้อง และเป็นการดีที่อัลลอฮฺได้ทรงสั่งใช้แก่เรา แต่เมื่อจะทำเช่นนั้นแล้ว จำเป็นที่จะต้องปฏิบัติตามแนวทางต่อไปนี้ : - หลีกห่างจากการเลียนแบบของผู้ปฏิเสธศรัทธาและประเพณีของพวกเขาในเรื่องดังกล่าว - อย่ากำหนดวันที่ซ้ำกันทุกปีเพราะกรณีนี้เป็นรูปแบบหนึ่งของบาปตามที่อธิบายไว้ก่อนหน้านี้ - เรียกร้องให้ผู้หญิงสวมฮิญาบ ที่ถูกต้อง การรู้สึกผูกยึดและติดอยู่กับเพศตรงข้ามเยี่ยงดังกล่าวจะไม่โดนกับหัวใจที่เต็มเปี่ยมไปด้วยความรักต่ออัลลอฮฺ แต่มันจะโดนกับหัวใจที่ว่างเปล่า อ่อนแอ และเพลี่ยงพล้ำ จากนั้นมันก็จะเข้าไปควบคุมหัวใจ ถ้าหากมันขยายใหญ่โตไปเรื่อยๆ มันอาจจะเข้าไปปกคลุมมีชัยชนะเหนือความรักต่ออัลลอฮฺ และนำเจ้าของไปสู่การตั้งภาคีต่ออัลลอฮฺได้ นี่เองคือที่มาของคำกล่าวที่ว่า "อารมณ์ใฝ่ต่ำนั้นคือความเคลื่อนไหวของหัวใจที่ว่างเปล่า" เพราะหัวใจนั้นเมื่อว่างเปล่าจากความรักต่ออัลลอฮฺผู้ทรงเมตตา ไม่มีการระลึกถึงพระองค์และปราศจากความอิ่มเอิบด้วยการเข้าเฝ้าและสยบยอมต่อถ้อยคำของพระองค์แล้วละก็ ย่อมต้องสาละวนอยู่กับการรักใคร่สตรีเพศ หลงอยู่กับรูปลักษณ์ และการฟังเพลง ชัยคุ้ลอิสลาม อิบนุตัยมียะฮฺ ได้กล่าวว่า : เมื่อหัวใจมีความรักต่ออัลลอฮฺและบริสุทธิ์ใจต่อพระองค์เพียงผู้เดียว ย่อมจะไม่ถูกทดสอบให้รักผู้อื่นนอกจากพระองค์ด้วยซ้ำ นับประสาอะไรกับการใคร่หลงงมงายอยู่ในความรัก และเมื่อถูกทดสอบให้ตกอยู่ในการหลงรักคนอื่น นั่นก็แสดงว่า เป็นเพราะการบกพร่องในความรักที่มีต่อพระองค์ ด้วยเหตุนี้ ท่านนบียูซุฟ ( อาลัยฮิสสลาม ) ซึ่งเป็นคนที่มีความรักและบริสุทธิ์ใจต่ออัลลอฮฺเพียงผู้เดียว จึงไม่ตกเป็นผู้ถูกทดสอบให้ใคร่หลงตกอยู่ในความรักเยี่ยงนั้น อัลลอฮฺ ( ซุบฮานาฮุวาตะอาลา ) ได้ตรัสว่า : ( الأية )..كَذَٰلِكَ لِنَصْرِفَ عَنْهُ السُّوءَ وَالْفَحْشَاءَ إِنَّهُ مِنْ عِبَادِنَا الْمُخْلَصِينَ (٢٤) ความว่า : ( อัลอายะฮฺ )..เช่นนั้นแหละเพื่อเราจะให้ความชั่วและการลวนลามห่างไกลจากเขา( ห่างไกลจากท่านนบียูซุฟ ) แท้จริงเขาคือคนหนึ่งในปวงบ่าวของเราที่สุจริต ( ซูเราะฮฺยูซุฟ อายะฮฺที่ 24 ) ส่วนภรรยาของเจ้าเมืองและสาวๆในเมืองนั้น พวกนางเป็นพวกที่ตั้งภาคีต่ออัลลอฮฺ ดังนั้นจึงตกอยู่ในบททดสอบแห่งความใคร่หลงดังกล่าว [ มัจมัวะอฺ อัลฟะตาวา เล่มที่ 10 หน้าที่ 135 ] ดังนั้นสิ่งที่จำเป็นเหนือมุสลิมก็คือ ต้องหาทางให้รอดพ้นแก่ตัวเองจากความหายนะนี้ ต้องไม่ละเลยที่จะปกป้องและดูแลตัวเองให้ปลอดภัย ถ้าหากเขาบกพร่องในเรื่องดังกล่าว แล้วกลับไปเดินบนเส้นทางแห่งความรักและเป็นแฟนด้วยการทอดสายตามองสิ่งที่หะรอมเป็นประจำ ฟังสิ่งที่หะรอมอยู่เสมอ ทำเป็นเรื่องเล็กน้อยกับการพูดคุยกับเพศตรงข้าม หรืออะไรในทำนองนั้น เป็นต้น แล้วเขาก็ตกอยู่ในหลุมพรางแห่งความรักและความใคร่หลงระหว่างชายหญิงอย่างไม่ถูกต้อง เขาก็ย่อมเป็นผู้ที่ทำบาปและต้องถูกลงโทษเนื่องด้วยพฤติกรรมนั้น มีผู้คนมากเท่าใดที่ไม่แยแสกับหลักการต่างๆที่เกี่ยวข้องกับพฤติกรรมดังกล่าว และคาดคิดว่าสามารถที่จะคุมตัวเองได้เมื่อใดที่เขาต้องการ หรือคิดไปเองว่าสามารถที่จะหยุดยั้งตัวเองได้โดยไม่ละเมิดขอบเขตที่ต้องห้าม สุดท้ายเมื่อโรคหลงรักที่ว่าซึมซับจนกลายเป็นนิสัยที่ยากจะสลัดพ้น เมื่อนั้นก็ไม่มีหมอหรือยาใดๆที่สามารถจะรักษาเขาได้อีก มีผู้หนึ่งกล่าวว่า : เมื่อบุรุษหนึ่งตกอยู่ในความรัก ครั้นเมื่อคิดจะพัก กลับมิอาจทำได้ เขาเห็นคลื่นลูกใหญ่ แต่ใจกลับคิดว่ามันเล็ก จนเมื่อมันเข้าประชิด เขาก็จมดิ่งมิหวนคืน อิบนุ้ลก็อยยิม ได้กล่าวไว้ว่า : เมื่อใดที่มีมูลเหตุเกิดขึ้นด้วยความสมัครใจของเขา หากมีผลกรรมใดๆตามมาหลังจากนั้นโดยที่เขาไม่ได้ตั้งใจก็ย่อมไม่ถูกอนุโลมให้ เมื่อเขาสร้างมูลเหตุที่ต้องห้ามแล้วส่งผลให้เกิดอาการมึนเมาเพราะเหตุที่ว่า เขาก็จะไม่ได้รับการอนุโลมในความผิดนั้น และเป็นที่ชัดเจนอย่างไม่ต้องสงสัยว่า การทอดสายตาและครุ่นคิดเรื่องรักๆใคร่ๆอยู่ตลอดเวลานั้นเปรียบเสมือนคนที่ดื่มของมึนเมา ดังนั้นจงอย่าได้กล่าวหาและต่อว่ามูลเหตุที่ว่านั้นอีกเลย [ เราเฎาะตุล มุหิบบีน หน้าที่ 147 ] ถ้าหากเขาพยายามที่จะหลีกเลี่ยงจากช่องทางต่างๆที่นำไปสู่โรคภัยนี้ด้วยการลดสายตาจากการมองสิ่งต้องห้าม ระงับหูไม่ให้ฟังสิ่งที่ไม่ถูกต้อง และเบี่ยงหัวใจให้ออกห่างจากความคิดฟุ้งซ่านที่ชัยฎอนพ่นเข้าไปในใจของเขา ทว่าเขาก็ยังได้รับผลกระทบจากเศษเสี้ยวของโรคที่ว่านี้ เนื่องเพราะการเผลอมองชั่ววูบ หรือเพราะการปฏิสัมพันธ์ที่ดั้งเดิมแล้วไม่ใช่สิ่งต้องห้าม จนกลายเป็นเหตุให้เขายึดติดอยู่กับผู้หญิงคนหนึ่งคนใด กรณีนี้เขาจะไม่บาป อินชาอัลลอฮฺ เพราะอัลลอฮฺได้ตรัสว่า : لَا يُكَلِّفُ اللَّهُ نَفْسًا إِلَّا وُسْعَهَا..( الأية ) (٢٨٦) ความว่า : อัลลอฮฺจะไม่ทรงบังคับใช้ชีวิตหนึ่งชีวิตใด เว้นแต่เท่าที่ชีวิตนั้นสามารถจะแบกรับได้ ( ซูเราะฮฺอัลบากอเราะฮฺ อายะฮฺที่ 286 ) ชัยคุ้ลอิสลาม อิบนุตัยมียะฮฺ ได้กล่าวว่า : ใครที่ไม่ปล่อยปะละเลยและไม่ด้ทำการล่วงละเมิด เขาก็ไม่ต้องรับผิดในสิ่งที่ประสบกับเขา [ มัจมัวะอฺ อัลฟะตาวา เล่มที่ 11 หน้าที่ 10 ] อิบนุ้ลก็อยยิม ยังได้กล่าวว่า : เมื่อความหลงรักเกิดขึ้นไม่ใช่ด้วยสาเหตุที่ผิด เจ้าของมันก็จะไม่ถูกตำหนิ เช่นคนที่รักภรรยาของเขาหรือรักทาสหญิงของเขาเอง แล้วเขามีเหตุจำเป็นต้องพรากจากนาง แต่ความรักของเขาต่อนางยังคงติดอยู่กับเขาไม่ยอมหาย เช่นนี้ เขาก็จะไม่ถูกตำหนิแต่ประการใด เช่นเดียวกับกรณีที่พลั้งไปมองโดยไม่ตั้งใจแล้วก็หันสายตาไปทางอื่น แต่มิอาจจะต้านทานความรู้สึกรักที่เข้ามาครอบงำในใจของเขาโดยที่เขาไม่ได้สมัครใจเลือกที่จะเป็นอย่างนั้น อย่างไรก็ตาม เป็นสิ่งจำเป็นที่เขาต้องพยายามต่อสู้และสลัดความรู้สึกดังกล่าวให้พ้นไปจากตัวเอง [ เราเฎาะตุล มุหิบบีน หน้าที่ 147 ] จำเป็นที่เขาต้องพยายามเยียวยาหัวใจด้วยการตัดขาดจากผลที่เกิดขึ้นเพราะคนรักของเขา และด้วยการเพิ่มความรักต่ออัลลอฮฺลงไปในหัวใจแทน ให้มีความรู้สึกเพียงพอกับความรักที่มีต่อพระองค์ และไม่ควรละอายที่จะขอคำปรึกษาจากคนที่มีปัญญาและความจริงใจในการให้คำปรึกษา หรืออาจจะเข้ารับการรักษาจากหมอหรือผู้เชี่ยวชาญทางจิต อิบนุตัยมียะฮฺ ได้กล่าวว่า : ถ้าหากโดนทดสอบให้มีอาการใคร่หลงอยู่ในความรัก แล้วเขาก็อดทนและปกป้องดูแลความบริสุทธิ์ของตน เขาก็ย่อมที่จะได้รับผลบุญเนื่องด้วยความยำเกรงของเขาต่ออัลลอฮฺ เพื่อไม่ให้เกิดมีการพูดที่ต้องห้ามขึ้นมา ไม่ว่าจะเป็นการร้องเรียนต่อมัคลูกหรือการเปิดเผยความเลวทรามต่ำช้า หรือการอ้อนวอนร้องขอจากคนรัก (สูเราะฮฺอัลบะเกาะเราะฮฺ อายะฮฺที่ 165) إِنَّ ٱلَّذِينَ هُم مِّنْ خَشْيَةِ رَبِّهِم مُّشْفِقُونَ ความว่า : แท้จริงบรรดาผู้ที่พวกเขาเป็นผู้มีจิตใจยำเกรงเนื่องจากความกลัวต่อพระเจ้าของพวกเขา (สูเราะฮฺอัลมุอฺมินูน อายะฮฺที่ 57) และอัลลอฮฺทรงกล่าวถึงทั้งสองสิ่งนี้พร้อมกันในอายะฮฺที่ว่า : إِنَّهُمْ كَانُوا۟ يُسَـٰرِعُونَ فِى ٱلْخَيْرَٰتِ وَيَدْعُونَنَا رَغَبًۭا وَرَهَبًۭا ۖ وَكَانُوا۟ لَنَا خَـٰشِعِينَ ความว่า : แท้จริงพวกเขารีบเร่งในการทำความดีและพวกเขาวิงวอนต่อเราด้วยความหวังและความกลัว (ต่อการลงโทษของเรา) และพวกเขาเป็นผู้ถ่อมตัวเกรงกลัวต่อเรา (สูเราะฮฺอัลอัมบิยาอ์ อายะฮฺที่ 90) เมื่อเข้าใจเช่นนี้แล้ว เราก็จะเข้าใจได้ว่าการอิบาดะฮฺนั้น จะถูกตอบรับเฉพาะจากมุสลิมที่ศรัทธาในอัลลอฮฺเพียงพระองค์เดียว(มีเตาฮีด) เท่านั้น ดังที่อัลลอฮฺตรัสว่า : وَقَدِمْنَآ إِلَىٰ مَا عَمِلُوا۟ مِنْ عَمَلٍۢ فَجَعَلْنَـٰهُ هَبَآءًۭ مَّنثُورًا ความว่า : และเราจะมุ่งสู่ส่วนหนึ่งของการงานที่พวกเขา(ผู้ปฏิเสธ) ได้ปฏิบัติไป แล้วเราจะทำให้มันไร้คุณค่ากลายเป็นละอองฝุ่นที่ปลิวว่อน (สูเราะฮฺอัลฟุรกอน อายะฮฺที่ 23) ในเศาะเฮียฮฺมุสลิม (หมายเลข 214) มีรายงานว่า ท่านหญิงอาอิชะฮฺ (เราะฎิยัลลอฮุอันฮา) กล่าวว่า : ฉันกล่าวว่า "โอ้ท่านเราะสูลุลลอฮฺ ในยุคญาฮิลียะฮฺ อิบนุญุดอาน เคยรักษาความสัมพันธ์ที่ดีกับเครือญาติและให้อาหารแก่คนยากจน การงานเหล่านี้จะเป็นประโยชน์แก่เขาหรือไม่?" ท่านตอบว่า "มันจะไม่เกิดประโยชน์ใดๆแก่เขาเลย เพราะเขาไม่เคยกล่าวว่า "โอ้อัลลอฮฺ โปรดอภัยโทษแก่บาปของข้าพระองค์ในวันแห่งการพิพากษาด้วยเถิด"' กล่าวคือ เขาไม่ได้ศรัทธาในการฟื้นคืนชีพหรือทำความดีโดยหวังที่จะพบอัลลอฮฺเลยยิ่งไปกว่านั้น การอิบาดะฮฺของมุสลิมจะไม่ได้ถูกตอบรับ เว้นแต่จะตรงตามเงื่อนไขพื้นฐานสองประการนี้ : 1. ความบริสุทธิ์(อิคลาศ) ในเจตนาที่มีต่ออัลลอฮฺ ซึ่งหมายความว่าเจตนาของบุคคลนั้น ไม่ว่าจะเป็นคำพูดหรือการกระทำของเขาทั้งหมด ทั้งที่ปรากฏจากภายนอกและภายใน จะต้องเป็นไปเพื่อแสวงหาความพอพระทัยจากอัลลอฮฺเท่านั้น 2. ต้องสอดคล้อง (มุวาฟะเกาะฮฺ) กับชะรีอะฮฺที่อัลลอฮฺทรงบัญญัติ และต้องไม่ทำการอิบาดะฮฺต่ออัลลอฮฺด้วยกับรูปแบบอื่นจากที่ถูกบัญญัติไว้ ซึ่งสิ่งนี้จะบรรลุผลได้ด้วยการปฏิบัติตามท่านนบี (ศ็อลลัลลอฮุอะลัยฮิวะสัลลัม) และสิ่งที่ท่านได้นำมา อีกทั้งต้องละทิ้งทุกสิ่งที่ขัดกับแนวทางของท่าน และต้องไม่อุตริอิบาดะฮฺใหม่ขึ้นมา โดยที่มันไม่ได้ถูกรายงานไว้ในหะดีษที่ถูกต้องจากท่านนบี (ศ็อลลัลลอฮุอะลัยฮิวะสัลลัม) หลักฐานสำหรับเงื่อนไขสองประการข้างต้นนี้ คืออายะฮฺที่ว่า : فَمَن كَانَ يَرْجُوا۟ لِقَآءَ رَبِّهِۦ فَلْيَعْمَلْ عَمَلًۭا صَـٰلِحًۭا وَلَا يُشْرِكْ بِعِبَادَةِ رَبِّهِۦٓ أَحَدًۢا ความว่า : ดังนั้นผู้ใดหวังที่จะพบกับพระผู้เป็นเจ้าของเขา ก็ให้เขาประกอบการงานที่ดี #QA ตาบิอีนและตาบิอิตตาบิอีนคือใคร? ถาม : ตาบิอีนและตาบิอิตตาบิอีนคือใคร? ตอบ : อัลฮัมดุลิลลาฮฺ ประการที่ 1 อัตตาบิอีน คือผู้ที่มาหลังจากยุคของท่านนบีมุฮัมมัด ﷺ และไม่ทันได้พบเจอกับท่าน ﷺ แต่พวกเขาได้พบกับบรรดาเศาะฮาบะฮฺของท่านนบี ﷺ และได้เรียนรู้จากพวกเขา ตาบิอิตตาบิอีน หรือ อัตบาอิตตาบิอีน คือผู้ที่ไม่ทันได้พบกับบรรดาเศาะฮาบะฮฺของท่านนบี ﷺ แต่ได้พบกับตาบิอีนและได้เรียนรู้จากพวกเขา ในทางด้านมุศเฏาะละหฺ อัลหะดีษ คำว่า ตาบิอีน คือบุคคลที่ได้พบเจอกับเศาะฮาบะฮฺ โดยไม่จำเป็นว่าพวกเขาจะต้องอยู่ด้วยกันเป็นเวลานานตามทัศนะที่ถูกต้อง ฉะนั้นใครก็ตามที่ได้พบเจอกับเศาะฮาบะฮฺและเสียชีวิตในฐานะมุสลิม ถือว่าเขาคือตาบิอีน และบางส่วนในหมู่พวกเขาก็อาจดีกว่าอีกบางส่วน ท่านอัลฮาฟิซ อิบนุฮะญัร (เราะฮิมะฮุลลอฮฺ) ได้กล่าวไว้ใน นุคบะตุลฟิกรฺ เล่มที่ 4 หน้าที่ 724) ว่า : ตาบิอีนคือผู้ที่ได้พบกับเศาะฮาบะฮฺ ท่านอิบนุกะษีร (เราะฮิมะฮุลลอฮฺ) ได้กล่าวว่า : อัลคอฏีบ อัลบัฆดาดีย์ กล่าวว่า "ตาบิอีนคือผู้ที่ได้พบเจอกับเศาะฮาบะฮฺ และจากคำพูดของท่านอัลฮากิม เราสามารถเข้าใจได้ว่าคำว่าตาบิอีสามารถถูกใช้กับผู้ที่ได้พบกับเศาะฮาบะฮฺและได้รายงานจากพวกเขาได้ แม้ว่าพวกเขาจะไม่ได้อยู่ด้วยกันเป็นเวลานานก็ตาม" ท่านอัลอิรอกี (เราะฮิมะฮุลลอฮฺ) กล่าวไว้ใน อัลฟียะฮฺ หน้า 66 ว่า : ตาบิอีนคือผู้ที่ได้พบเจอกับผู้ที่เคยอยู่กับท่านนบี ตาบิอิตตาบิอีน คือผู้ที่ได้พบกับตาบิอีน แต่ไม่ได้พบกับเศาะฮาบะฮฺ (เราะฎิยัลลอฮุอันฮุม) บางส่วนจากตาบิอีน เช่น ท่านสะอีด อิบนุลมุสัยยิบ, อุรวะฮฺ อิบนุ อัสสุบัยรฺ, อัลฮะสัน อัลบัศรี, มุฮาญิด อิบนุ ญับรฺ, สะอีด อิบนุญุบัยรฺ, อิกริมะฮ์ ผู้รับใช้ของอิบนุอับบาส และนาฟิอฺ ผู้รับใช้ของอิบนุอุมัร บางส่วนจากตาบิอิตตาบิอีน เช่น เช่น ท่านสุฟยาน อัษเษารี, อิมามมาลิก, รอบีอะฮ์, อิบนุฮุรมุซ, อัลฮะสัน อิบนุศอเลียะฮฺ, อับดุลลอฮ์ อิบนุลฮะซัน, อิบนุอะบีลัยลา, อิบนุชุบรุมะฮ์ และอัลเอาซาอี ประการที่ 2 ท่านอิมามอัลบุคอรี (3651) และอิมามมุสลิม (2533) รายงานจากอิบนุมัสอูด (เราะฎิยัลลอฮุอันฮุ) ว่าท่านนบี ﷺ กล่าวว่า : خَيْرُ النَّاسِ قَرْنِي ، ثُمَّ الَّذِينَ يَلُونَهُمْ ، ثُمَّ الَّذِينَ يَلُونَهُمْ ، ثُمَّ يَجِيءُ أَقْوَامٌ تَسْبِقُ شَهَادَةُ أَحَدِهِمْ يَمِينَهُ، وَيَمِينُهُ شَهَادَتَهُ ความว่า : มนุษย์ที่ดีที่สุดคือกลุ่มชนในยุคสมัยของฉัน (เราะฮิมะฮุลลอฮฺ) กล่าวว่า : ทัศนะที่ถูกต้องคือกลุ่มชนที่อยู่ในยุคสมัยของท่านนบีคือเศาะฮาบะฮฺ กลุ่มชนรุ่นที่สองคือตาบิอีน และกลุ่มชนรุ่นที่สามคือตาบิอิตตาบิอีน (ชัรหฺ อันนะวะวี อะลามุสลิม เล่มที่ 16 หน้าที่ 85) ท่านอัลฮาฟิซ อิบนุฮะญัร (เราะฮิมะฮุลลอฮฺ) กล่าวว่า : คำว่า "ผู้กลุ่มชนที่มาถัดจากพวกเขา" หมายถึงกลุ่มชนในรุ่นที่ตามมาหลังพวกเขา นั่นคือตาบิอีน และ "กลุ่มชนที่มาถัดจากพวกเขาอีกที" หมายถึงตาบิอิตตาบิอีน (ฟัตฮุลบารี เล่มที่ 7 หน้าที่ 6) ท่านมุลลา อะลี อัลกอรี (เราะฮิมะฮุลลอฮฺ) กล่าวว่า : อัสสุยูตี กล่าวว่า ทัศนะที่ถูกต้องคือคำว่า อัลก็อรนฺ (ยุคหรือรุ่น) ยุคของตาบิอีนนั้นสิ้นสุดเมื่อประมาณปีฮ.ศ. 170 และยุคตาบิอิตตาบิอีนสิ้นสุดเมื่อประมาณปีฮ.ศ. 220 โดยในช่วงเวลานี้อุตริกรรม (บิดอะฮฺ) ทางความเชื่อได้ปรากฏขึ้นอย่างมากมาย #QA เมื่อเราเตาบะฮฺ จำเป็นต้องเอ่ยถึงบาปทั้งหมดหรือไม่? ถาม : คนที่กลับเนื้อกลับตัว (เตาบะฮฺ) จากการชิริก จำเป็นต้องกล่าวเอ่ยถึงการทำชิริกทุกอย่างที่เขาได้เคยกระทำหรือไม่? หรือว่านั่นเป็นกลอุบายของชัยฏอนที่มุ่งจะขัดขวางเขาจากการเตาบะฮฺและเข้ารับอิสลาม? หรือว่าการปฏิญาณตน (ชะฮาดะฮฺ) ทั้งสองเพียงพอแล้วสำหรับการเข้ารับอิสลาม? ตอบ : อัลฮัมดุลิลลาฮฺ คนที่ล่วงล้ำไปสู่ชิริกหรือบาปอื่น ๆ ไม่จำเป็นต้องกล่าวเอ่ยถึงบาปทุกอย่างที่ตนเองได้กระทำลงไปแต่อย่างใด การที่เขาจะเตาบะฮฺอย่างจริงใจต่ออัลลอฮ์ ตะอาลา โดยเป็นการเตาบะฮฺในลักษณะภาพรวม (อาม) พร้อมกับปฏิบัติครบตามเงื่อนไขที่จำเป็นสำหรับการเตาบะฮฺ ก็ถือว่าเพียงพอแล้วที่จะทำให้การเตาบะฮฺของเขาได้รับการตอบรับ ท่านชัยคุลอิสลาม อิบนุตัยมียะฮ์ ได้กล่าวในอัลฟะตาวา อัลกุบรอ (เล่มที่ 5 หน้าที่ 281) ว่า : “หากใครก็ตามเตาบะฮฺในลักษณะภาพรวม (อาม) การเตาบะฮฺนี้จะนำไปสู่การอภัยโทษแก่บาปทั้งหมดของเขา แม้ว่าเขาจะไม่ได้กล่าวเอ่ยถึงบาปแต่ละอย่างเป็นการเฉพาะ ยกเว้นในกรณีที่มีเหตุที่เขากล่าวถึงบาปประการหนึ่งหรือมากกว่านั้นเป็นการเฉพาะ (ซึ่งจะทำให้การอภัยโทษนั้นไม่ครอบคลุมถึงบาปเหล่านี้ - ผู้แปล) เช่น หากมีบาปประการหนึ่งที่เมื่อเขานึกขึ้น แล้วเขาไม่ต้องการที่จะเตาบะฮฺจากมันเพราะเขายังมีความต้องการที่จะทำบาปเหล่านั้นอีก หรือเพราะเขาเชื่อว่าการกระทำนั้นเป็นสิ่งที่ดีและไม่ใช่ความชั่ว ดังนั้นหากมีบาปประการหนึ่งที่เขานึกคิดไว้ว่าไม่ต้องการเตาบะฮฺจากบาปนั้นแล้ว มันก็จะไม่รวมอยู่ในการเตาบะฮฺ (ในลักษณะภาพรวม) นั้น แต่สำหรับบาปที่เมื่อเขานึกถึงในขณะที่กล่าวคำเตาบะฮฺในลักษณะภาพรวม แล้วเขาต้องการเตาบะฮฺจากมัน บาปเหล่านั้นจะถูกรวมอยู่ในเตาบะฮฺลักษณะภาพรวมนี้ด้วย” อิมามอิบนุลก็อยยิม ได้กล่าวในมะดาริจญ์ อัสสาลิกีน (เล่มที่ 1 หน้าที่ 283) ว่า : “การเร่งรีบในการเตาบะฮฺจากบาปเป็นสิ่งจำเป็นที่ต้องกระทำทันทีและไม่อนุญาตให้ล่าช้า หากใครก็ตามล่าช้าในการเตาบะฮฺ ถือว่าเขาได้สร้างบาปอีกประการหนึ่งจากการการล่าช้านั้น ดังนั้นเมื่อเขาเตาบะฮฺจากบาป (โดยล่าช้า) แล้ว จะยังคงมีการเตาบะฮฺอีกอย่างหนึ่งที่จำเป็นต้องกระทำสำหรับเขา คือการเตาบะฮฺจากการล่าช้าในการเตาบะฮฺ ซึ่งน้อยครั้งนักที่ผู้เตาบะฮฺจะคิดถึงเรื่องนี้ เขามักจะคิดว่าเมื่อเขาเตาบะฮฺจากบาปแล้ว ไม่มีสิ่งอื่นใดที่จำเป็นสำหรับเขาอีก ทั้งที่ยังคงเหลือการเตาบะฮฺจากการล่าช้าการเตาบะฮฺ ซึ่งไม่มีสิ่งใดที่จะช่วยคนคนหนึ่งจากสภาพนี้ได้นอกจากการเตาบะฮฺในลักษณะภาพรวม (อาม) ที่ครอบคลุมถึงบาปทั้งหมดที่เขารู้ตัวและบาปที่เขาไม่รู้ตัว เพราะบาปที่เขาไม่รู้ตัวนั้นมีจำนวนมากกว่าบาปที่เขารู้ตัวเสียอีก การที่คนคนหนึ่งไม่รู้ถึงบาปของเขา จะไม่เป็นข้อแก้ตัวให้เขานั้นพ้นจากการถูกสอบสวนในบาปเหล่านั้น หากเขาสามารถแสวงหาความรู้ได้ เพราะเขากำลังทำบาปด้วยการละเลยการแสวงหาความรู้และการปฏิบัติที่ถูกต้อง ดังนั้นบาปในกรณีนี้จึงยิ่งทวีความรุนแรงขึ้น ในเศาะเฮียฮฺอิบนุฮิบบาน มีรายงานว่า ท่านนบี ﷺ กล่าวว่า : "การตั้งภาคีในประชาชาตินี้นั้นแนบเนียนยิ่งกว่าเสียงฝีเท้าของมดเสียอีก" ท่านอบูบักรฺ เราะฎิยัลลอฮุอันฮฺ กล่าวว่า "เราจะกำจัดมันได้อย่างไร โอ้เราะซูลลุลลอฮฺ?" ท่านกล่าวว่า "ด้วยการกล่าวว่า اللَّهُمَّ #QA ถาม : ใครคือกลุ่มหลงผิดในยุคต้นของอิสลาม? ตอบ : บรรดาเศาะหาบะฮฺ (เราะฎิยัลลอฮุอันฮุม) คือกลุ่มคนที่เผชิญกับฟิตนะฮฺน้อยที่สุดเมื่อเทียบกับกลุ่มคนที่มาภายหลังพวกท่าน เพราะยิ่งเวลาผ่านไปหลังจากยุคของท่านนบี ﷺ ความแตกแยกและความขัดแย้งก็ยิ่งเพิ่มมากขึ้น ในช่วงการปกครองของท่านอุษมาน (เราะฎิยัลลอฮุอันฮฺ) ยังไม่ปรากฏบิดอะฮฺ (การอุตริกรรม) อย่างเปิดเผยชัดเจน แต่ทว่าเมื่อท่านอุษมานถูกสังหาร ก็เริ่มเกิดความแตกแยกในหมู่ผู้คนและได้ปรากฏบิดอะฮฺขึ้นสองประการ ซึ่งเป็นสิ่งที่อยู่ขั้วตรงข้ามกัน - บิดอะฮฺของพวกเคาะวาริจญ์ ซึ่งคือกลุ่มที่ตัดสิน (ตักฟีร) ท่านอะลีว่าเป็นกาฟิร - บิดอะฮฺของพวกรอฟิเฎาะฮฺ ซึ่งคือกลุ่มที่อ้างว่าท่านอะลีเป็นอิมามอย่างสมบูรณ์และไร้ซึ่งความผิด บ้างก็อ้างว่าท่านเป็นนบี หรือถึงขั้นอ้างว่าท่านเทียบเท่ากับอัลลอฮฺ! ต่อมาในช่วงที่บรรดาเศาะหาบะฮฺรุ่นสุดท้ายยังมีชีวิตอยู่ ในระหว่างการปกครองของอิบนุซซุบัยร์และอับดุลมาลิก ก็ปรากฏบิดอะฮฺต่อมา คือ - พวกมุรญิอะฮฺ - พวกเกาะดะรียะฮฺ จากนั้นในช่วงแรกของยุคตาบิอีน ที่ใกล้ถึงจุดสิ้นสุดการปกครองของราชวงศ์อุมัยยะฮฺ ก็ปรากฏบิดอะฮฺตัวใหม่อีก - พวกญะฮฺมียะฮฺ อัล-มุอัฏฏิละฮฺ - พวกมุชับบิฮะฮฺ อัล-มุมัษษิละฮฺ โดยบิดอะฮฺเหล่านี้ยังไม่เคยเกิดขึ้นในสมัยของบรรดาเศาะหาบะฮฺ ------------------------------------------------------ ชัยคุลอิสลาม อิบนุตัยมิยะฮฺ มินฮาญุซซุนนะฮ์ อันนะบะวียะฮ์ เล่มที่ 6 หน้าที่ 231 แปลเป็นภาษาอังกฤษ : อบูฮาติม มุฮัมมัด ฟารูก #ใครคือกลุ่มหลงผิดในยุคต้นของอิสลาม #ถามตอบ #talib #talibclub #islam #muslim #salafi #salafithailand #ศาสนาอิสลาม #มุสลิม #ซะลาฟีย์ #แนวทางสลัฟ

ภรรยาของอับดุลลอฮฺ อิบนุ มัสอู๊ด รายงานว่า ท่านร่อซูลุลลอฮฺ ﷺ ได้กล่าวแก่พวกนางว่า: “หากคนหนึ่งในพวกเธอไปมัสยิด ก็อย่าใช้น้ำหอม” (บันทึกโดยมุสลิม เลขที่ 443) เมื่อท่านนบี ﷺ ห้ามสตรีออกไปยังมัสยิดโดยใช้น้ำหอม เนื่องจากโดยปกติแล้วบุรุษย่อมได้กลิ่นน้ำหอมจาก ความใกล้ชิดและการที่ไม่มีสิ่งกีดขวางระหว่างชายกับหญิง ดังนั้นจึง ยิ่งมีน้ำหนักมากขึ้นไปอีก ที่จะกล่าวว่า ดังที่เห็นได้อย่างชัดเจน” 5.กรณีสตรีใช้น้ำหอมโดยเชื่อว่ากลิ่นของนางจะไม่ฟุ้งกระจายไปถึงผู้อื่น เมื่อสตรีมุสลิมใช้น้ำหอม โดยมีความเห็นเป็นไปได้สูงว่า กลิ่นน้ำหอมของเธอจะไม่ไปถึงผู้อื่น หรือกรณีที่เธอไปมัสยิดโดยรถยนต์ และลงรถตรงทางเข้าห้องละหมาด สำหรับสตรีซึ่งแยกออกจากบุรุษโดยสิ้นเชิง จากนั้นกลับขึ้นรถทันทีโดยมิได้เดินผ่านท้องถนน หรือสถานการณ์อื่นใดในลักษณะเดียวกัน ซึ่งสตรี (โรลออน)" หรือ "น้ำยาปรับผ้านุ่ม" ด้วย หากเจตนาใน การใช้คือเพื่อขจัดกลิ่นไม่พึงประสงค์และเพื่อสุขอนามัย โดยที่กลิ่นนั้นหอมเพียงอ่อนๆ ไม่รุนแรงหรือฟุ้ง กระจายจนเป็นที่สังเกตของบุรุษเมื่อเดินผ่าน
